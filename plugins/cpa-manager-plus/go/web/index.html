在 CPA plugins.configs.cpa-manager-plus 中配置：
manager_base_url — Manager 地址（默认 127.0.0.1:18317）
management_key — Plus Manager 的 Bearer（仅服务端访问 18317 时使用；本机无鉴权可省略）
勿将 CPA secret-key 写入插件配置。
